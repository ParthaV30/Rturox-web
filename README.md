# Rturox

A premium tech studio website built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Single-page application with smooth scrolling
- Dark premium aesthetic
- Responsive design
- Contact form with Resend API integration
- SEO optimized

## Environment Variables

Create a `.env.local` file with:

```
RESEND_API_KEY=your_resend_api_key
CONTACT_EMAIL=hello@rturox.com
```

## Deploy

Deploy on Vercel or any Next.js compatible platform.
